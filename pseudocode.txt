1. Sum

Pseudocode:

for i from 0 to n-1:
    for j from i+1 to n-1:
        if arr[i] + arr[j] == target:
            return i, j

2. Binary Number

Pseudocode:

low = 0
high = n - 1
while low <= high:
    mid = (low + high) / 2
    if arr[mid] == target:
        return mid
    else if arr[mid] < target:
        low = mid + 1
    else:
        high = mid - 1

4.Longest Substring Without Repeating Characters




Pseudocode:

start = 0, maxLen = 0
map[256] = {-1}
for end from 0 to n-1:
    if map[s[end]] >= start:
        start = map[s[end]] + 1
    map[s[end]] = end
    maxLen = max(maxLen, end - start + 1)
